GovTech Maturity Index metadata¶

Metadata for a selection of performance indicators from the World Bank GovTech Maturity Index. 2025. Source: https://datacatalog.worldbank.org/search/dataset/0037889/govtech-datasetCloud

Shared cloud platform¶

Field Value
Chapter GovTech Maturity
Variable Id govtech-cloud-1
Definition Is there a shared cloud platform available for all government entities?
Format 0=Unknown/n.a;1=No;2=Only cloud strategy/policy (no platform yet);3=Yes (platform in use)
Other Metadata

Cloud platform type¶

Field Value
Chapter GovTech Maturity
Variable Id govtech-cloud-1.4
Definition Type of cloud platform established
Format 0=Unkown/n.a.;1=Public (Commercial);2=Private (Government);3=Hybrid
Other Metadata Type of cloud > Public: Commercial operators; outside/in country; Private: Operated by government entities; on-premise, 3= Hybrid (combination of public \& private)
