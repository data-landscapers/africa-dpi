# Third party datasets

To provide a broader context to DPI developments, selections of indicators have been included from five well-established datasets:

- <a href="https://datacatalog.worldbank.org/search/dataset/0037889/govtech-dataset" target="_blank" rel="noopener noreferrer">The World Bank's GovTech Maturity Index, 2025</a>  
- <a href="https://www.worldbank.org/en/programs/statistical-performance-indicators" target="_blank" rel="noopener noreferrer">The World Bank's Statistical Performance Indicators, 2023</a>  
- <a href="https://odin.opendatawatch.com/data" target="_blank" rel="noopener noreferrer">Open Data Watch's Open Data Inventory, 2024</a>  
- <a href="https://datacatalog.worldbank.org/search/dataset/0037889/govtech-dataset" target="_blank" rel="noopener noreferrer">The Mo Ibrahim Index of African Governance, 2023</a>  

Metadata from these sources has been provided in the same format as the Perplexity approach.
