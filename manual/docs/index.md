# The Africa DPI Dashboard

The Africa Digital Public Infrastructure Continental Dashboard has been built as part of Research ICT Africa's **Africa Digital Public Infrastructure Study** for the African Development Bank.

## Navigation

-   The dashboard visualises ten datasets on the map of Africa. These are accessed from the buttons at the top of the page.
-   Each dataset is sub-divided into a number of **SECTIONS** which can be accessed from the dropdown top left above the map.
-   Each section contains a number of **VARIABLES** which can be accessed from the dropdown top right above the map.
-   Once a variable has been selected its **definition** can be seen in the text above the map. More details for each variable can be found in the metadata section of this manual (via the navigation sidebar)
-   All variables have been coded into a ‘traffic light’ scale with values from 1 up to 5. The description of these **‘scores’** can be seen below the definition. (The value of 0 is reserved for unknown data throughout.)
-   When a country is clicked **comments and sources** for all variables in the selected SECTION are displayed in the right-hand side panel.

## Content

**Collected for this project:**

Five datasets have been collected and curated using AI tools described in the [continental data collection methodology page](https://data-landscapers.com/africa-dpi/manual/site/methodology/) of this manual.

-   ICT Infrastructure
-   Data Exchange
-   Digital Id
-   Digital Pay
-   Registries

**Curated from third-party datasets**

The other five datasets have been curated and adjusted to our ‘traffic light’ methodology from selected variables drawn from:

-   The Carnegie Endowment for International Peace [**Africa Technology Policy Tracker**](https://carnegieendowment.org/features/africa-digital-regulations)
-   The World Bank [**GovTech Maturity Index**](https://www.worldbank.org/en/programs/govtech/gtmi)
-   The Mo Ibrahim Foundation [**Index of African Governance**](https://iiag.online/)
-   The Open Data Watch [**Open Data Inventory**](https://odin.opendatawatch.com/)
-   The World Bank [**Statistical Performance Indicators**](https://www.worldbank.org/en/programs/statistical-performance-indicators)
